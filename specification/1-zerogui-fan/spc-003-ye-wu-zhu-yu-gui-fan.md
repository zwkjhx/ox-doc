# 业务术语规范

业务术语规范用语定义一些固定的业务层面的含义规范，方便开发人员统一理解。

## 1. API方法定义

API方法定义描述了在Agent/Api两种Java文件中方法的名字。

| 项 | 含义 |
| :--- | :--- |
| get | 读取单条记录的专用前缀，只用于描述按照 id = value的方式查询单条记录的结果。 |
| post | 创建记录专用方法 |
| put | 更新记录专用方法 |
| delete | 删除记录专用方法 |

* API中的标准方法定义和HTTP Method对应，使得开发人员不用去思考POST/PUT究竟哪个规则用于更新。
